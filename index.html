📱 What is BuzzBit-Mind?

BuzzBit-Mind is an end-to-end automated video generation pipeline that creates educational short-form content about financial psychology and behavioral finance. Every day, the system produces videos for YouTube Shorts, TikTok, and YouTube Long-form — in 6 languages (French, English, Spanish, Arabic, Hindi, Chinese).

🔧 Full Pipeline Architecture

📰 1. News & Theme Selection

Web scraping + Gemini Grounding → extracts evergreen psychological hooks from current events.

NewsAPI Gemini 2.5 Flash Web Search

🧠 2. LLM Script Generation

Mistral Small 4 / GPT-5.4 mini / Gemini → generates 9-10 segments scripts with strict word count (13-16 words/segment) and empathy injection.

Mistral Small 4 GPT-5.4-mini Chain of Thought

🎙️ 3. AI Voiceover

Voxtral Mistral (Studio quality, 44.1kHz) → natural voices in 6 languages.

Voxtral Mistral Goggle studio 44.1kHz Mono

🎬 4. Dynamic B-roll

Pexels/Pixabay integration → automatic video clips matching visual prompts.

Pexels API Semantic Search

📝 5. Subtitles

Whisper + forced alignment → word-level timing, multi-language subtitles.

Faster-Whisper FFmpeg

🎨 6. Thumbnails

Stable Diffusion + Mistral selection → 4 variants generated, human validation via Telegram bot.

Stable Diffusion LLM Selection

🤖 7. Telegram Bot

Human-in-the-loop validation → approves thumbnails, schedules publications, monitors comments.

Telegraf Webhooks

📤 8. TikTok API

Uploads videos directly to TikTok drafts. No manual intervention.

TikTok API OAuth 2.0

🔄 Step-by-Step Automation Flow

1
News Ingestion — Daily news fetch → Gemini extracts evergreen psychological concepts.
2
Theme Selection — Random pick from 210+ behavioral finance themes (Foundation, Momentum, Mastery levels).
3
Script Generation — LLM writes script respecting:
• 9-10 segments (Foundation=9, Momentum/Mastery=10)
• 13-16 words per segment
• Empathy phrase injected before CTA
• Logic connectors (unique per segment)
• JSON output format
4
Audio Generation — Voxtral (Mistral) or Google tts produces voiceover at 44.1kHz + Whisper creates word-level timestamps.
5
B-roll Assembly — Semantic search for clips matching visual prompts → downloaded and concatenated.
6
Video Rendering — Remotion compositing (audio + b-roll + subtitles + thumbnail).
7
Human Validation — Telegram bot sends 4 thumbnails for approval.
8
Publication — Video uploaded to TikTok API (draft) + YouTube (private).
