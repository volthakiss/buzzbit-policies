Politique de confidentialité de BuzzBit

Dernière mise à jour : 19 mars 2026

1. Collecte des données

Nous collectons les informations suivantes lorsque vous utilisez notre Service :

  • Données d'authentification : tokens OAuth pour YouTube et TikTok (uniquement pour permettre les uploads)
  • Contenu généré : les scripts, titres et descriptions que vous produisez via notre outil
  • Données d'utilisation : logs d'utilisation pour améliorer le service

2. Utilisation des données

Vos données sont utilisées uniquement pour :

  • Fournir et maintenir le Service
  • Améliorer et personnaliser votre expérience
  • Communiquer avec vous concernant le Service

3. Partage des données

Nous ne vendons pas vos données personnelles. Nous ne partageons vos données qu'avec :

  • Les plateformes (YouTube, TikTok) via leurs API, lorsque vous autorisez l'upload
  • Les prestataires techniques (hébergement, base de données) nécessaires au fonctionnement du Service

4. Hébergement et sécurité

Ce site est hébergé sur GitHub Pages. GitHub peut collecter des adresses IP et des logs de visite conformément à leur déclaration de confidentialité.

Nous mettons en œuvre des mesures de sécurité pour protéger vos données contre tout accès non autorisé.

5. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès à vos données
  • Droit de rectification
  • Droit à l'effacement
  • Droit à la limitation du traitement

Pour exercer ces droits, contactez-nous à l'adresse ci-dessous.

6. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité. Nous vous encourageons à consulter régulièrement cette page.
